This is a little excercise to bring something like [GNU Nano](https://en.wikipedia.org/wiki/GNU_nano) to picocalc using Micropython on Picoware.

* curses for desktop, picoware will need a translation layer
* features
  * cut-paste
  * loading - saving text
  * text search
  * UTF-8 encoding (mainly using ASCII anyway)

* optional features
  * undo-redo
  * regex
  * launch
  * paste text files
